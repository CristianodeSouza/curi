# CURI — Dashboard Executivo de Atendimentos

Um dashboard responsivo desenvolvido para gerenciar e visualizar dados de atendimentos do sistema CURI (Curicuriari).

## 📋 Características

- **Dashboard Responsivo**: Interface adaptada para desktop, tablet e mobile
- - **KPIs em Tempo Real**:
  -   - Total de Atendimentos
      -   - Atendimentos Em Aberto
          -   - Atendimentos Resolvidos
              -   - Taxa de Resolução
                  - - **Design Moderno**: UI/UX otimizada com tema corporativo
                    - - **Topbar Responsiva**: Menu mobile com funcionalidades completas
                     
                      - ## 🛠️ Tecnologias
                     
                      - - **Frontend**: HTML5, CSS3, JavaScript
                        - - **Hospedagem**: GitHub Pages
                          - - **Dados**: Google Sheets (via MCP Google Drive)
                           
                            - ## 📁 Estrutura do Projeto
                           
                            - ```
                              curi/
